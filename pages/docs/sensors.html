This system supports multiple sensor drivers, all following the same interface. Each sensor returns a dictionary of canonical telemetry keys.

Sensors

Each sensor driver in the system returns a dictionary of canonical telemetry keys. All sensors inherit from the same base class and follow identical read semantics.

DHT22

Located in sensors/dht22.py

  • temperature_c
  • humidity
  • GPIO pin from config

DS18B20

Located in sensors/ds18b20.py

  • temperature_c
  • Reads from /sys/bus/w1/devices

Water Flow Sensor

Located in sensors/water_flow.py

  • flow_l_min
  • Smoothed and instant readings
  • Pigpio based timing
