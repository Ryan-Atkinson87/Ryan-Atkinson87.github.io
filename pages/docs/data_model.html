Data Model

This page defines all canonical telemetry keys used by Trive AquaSense. Each sensor driver returns a dictionary of values using consistent names and units. The telemetry collector forwards these values unchanged to ThingsBoard.

Telemetry Structure

Every message sent to ThingsBoard follows this structure:

{
  "ts": 1730022000000,
  "values": {
    "air_temperature": 24.1,
    "air_humidity": 52.3
  }
}

ts is always a UTC timestamp in milliseconds. values is the flattened telemetry dictionary produced by the sensor.

Canonical Telemetry Keys

DHT22

  • air_temperature — Degrees Celsius, float
  • air_humidity — Percent relative humidity, float

DS18B20 (1-Wire Temperature)

  • water_temperature — Degrees Celsius, float

Water Flow Sensor

  • flow_instant — Instant flow rate in liters per minute, float
  • flow_smoothed — Short term smoothed average, float

Attributes

Attributes are sent on startup or reconnect and describe the device.

  • ip_address
  • mac_address
  • device_name — From config.json
